# AirSlate PC Server

AirSlate PC Server 是 AirSlate 的 Windows 端配套程序。
它用于接收鸿蒙端应用 **AirSlate** 传来的笔输入与手势操作，让平板手写内容在电脑上直接生效。

> 鸿蒙端应用名为 **AirSlate**，需等待后续上架。

---

## 这是什么

如果你希望把鸿蒙设备当作电脑上的手写输入面板来使用，需要同时准备：

- Windows 电脑上的 **AirSlate PC Server**
- 鸿蒙设备上的 **AirSlate**

连接成功后，你可以在电脑端：

- 选择把输入映射到哪一块显示器
- 调整压感曲线，让下笔更轻或更扎实
- 使用快捷键预设
- 自定义手势、按键和径向菜单

## 使用前准备

### 电脑端

- Windows x64 系统
- 可正常运行 WebView2
- 与鸿蒙设备处于**同一局域网**

### 移动端

- 安装鸿蒙端应用 **AirSlate**
- 与电脑连接到同一网络

> 建议先确认电脑和鸿蒙设备连接的是同一个 Wi‑Fi / 局域网。

## 下载与安装

发布页通常会提供两种版本：

### 1. 安装版

文件示例：`AirSlate PC Server_1.4.10_x64.msi`

适合希望正常安装、后续直接使用的用户。

### 2. 便携版

文件示例：`AirSlate PC Server_1.4.10_x64_portable.zip`

适合希望解压即用、不写入常规安装位置的用户。

### 安装建议

- **大多数用户**：推荐安装版
- **临时使用 / 不想安装**：选择便携版

## 快速开始

### 第一步：启动 PC 端程序

打开 **AirSlate PC Server** 后，默认先进入“连接”页。

你会看到：

- 当前电脑的局域网 IPv4 地址
- 当前选择的显示器
- 压感曲线调节区域

![AirSlate PC Server 主界面总览](https://github.com/2doright/airslate-pc-server/blob/main/images/01-overview.jpg)

### 第二步：在鸿蒙端输入电脑 IP

在鸿蒙端 **AirSlate** 中，输入电脑当前显示的局域网 IPv4 地址并连接。

如果电脑上显示了多个地址，优先尝试列表中当前网络环境对应的那个地址。

![IP](https://github.com/2doright/airslate-pc-server/blob/main/images/02-ip.jpg)

### 第三步：确认映射显示器

如果你的电脑连接了多块屏幕，请先在 PC 端选择要映射的显示器。

![显示器](https://github.com/2doright/airslate-pc-server/blob/main/images/03-显示器.jpg)

建议：

- 单显示器：保持默认即可
- 双屏 / 多屏：先选定目标屏幕再开始使用

### 第四步：开始书写或触发手势

连接完成后，即可使用鸿蒙端进行书写、悬浮、点击或手势触发。

> 当前同一时间建议只连接一台 AirSlate 设备。

## 连接页说明

### 1. 局域网 IPv4 地址

这里显示的是当前电脑在局域网中的可连接地址。

如果这里没有显示地址，通常说明：

- 电脑未连接网络
- 当前网络不提供可用局域网地址
- 网络刚切换，建议点击刷新

### 2. 显示器选择

你可以在这里决定笔输入落到哪一块屏幕上。

适合这些场景：

- 副屏专门作为绘图屏使用
- 外接显示器作为主工作区
- 演示或扩展桌面时快速切换输入目标

### 3. 压感曲线

压感曲线用于调整“手上施加的压力”和“电脑端最终输出的压力”之间的关系。

预设包括：

- **线性**：输入输出基本一致
- **轻柔**：轻压就能更容易出笔
- **扎实**：需要更明确的压力才有更强输出
- **S 型**：中段变化更明显，适合需要层次感的场景

你也可以直接拖动曲线控制点进行微调。

![压感曲线调节](https://github.com/2doright/airslate-pc-server/blob/main/images/03-pressure-curve.jpg)

如果你不确定该选哪一个，可以从 **线性** 开始，再按手感慢慢调整。

## 快捷键页说明

快捷键页用于管理手势和按键映射。

你可以在这里：

- 切换当前预设
- 新建自己的预设
- 恢复当前预设的默认值
- 录入新的快捷键组合
- 配置径向菜单外环与内环行为

### 1. 预设管理

- **当前预设**：切换正在使用的方案
- **新建预设**：以当前方案为基础创建自己的版本
- **恢复默认**：把当前预设恢复到默认状态

如果你会在不同软件之间切换使用，建议按软件分别建立预设。

例如：

- 绘画软件一套
- 笔记软件一套
- 修图软件一套

![预设](https://github.com/2doright/airslate-pc-server/blob/main/images/04-preset.jpg)

### 2. 录入按键

在可编辑项上点击后，程序会进入录键状态。

操作方式：

- 按下你想设置的按键或组合键
- 松开后完成录入
- 按 `Esc` 可取消本次录入

### 3. 分类区域

当前支持的映射类别包括：

- **笔**：例如挤压、笔双击
- **点击**：双指 / 三指 / 四指点击
- **平移**
- **捏合**
- **旋转**
- **速划（单指）**
- **长按**

![分类区域](https://github.com/2doright/airslate-pc-server/blob/main/images/05-area.jpg)

---

## 径向菜单说明

双指平移支持呼出径向菜单。

它包含两部分：

- **内环**：固定方向槽位，可直接交换位置
- **外环**：支持分别录入不同方向的按键组合

你还可以控制：

- 是否启用内环
- 外环每个方向对应什么快捷键

![径向菜单设置](https://github.com/2doright/airslate-pc-server/blob/main/images/05-radial-menu.jpg)

如果关闭内环，双指划动会更直接地作用于外环。

## 默认使用建议

如果你是第一次使用，推荐按下面的顺序设置：

1. 先确认连接正常
2. 先选好目标显示器
3. 先用默认预设体验一轮
4. 再根据自己常用软件微调快捷键
5. 最后再微调压感曲线

这样最容易判断“是连接问题、手感问题，还是快捷键映射问题”。

---

## 开机启动

程序支持开机启动。

启用后，系统启动时会自动运行 AirSlate PC Server；自动启动时程序会仅驻留在系统托盘。

如果你每天都会用，建议打开。

## 常见问题

### 电脑上没有显示 IPv4 地址怎么办？

先检查：

- 电脑是否已经联网
- 是否连接到了局域网
- 是否刚切换过网络

可先点击刷新，再重新查看。

### 连接不上怎么办？

请优先检查：

- 电脑和鸿蒙设备是否在同一局域网
- 鸿蒙端输入的 IP 是否与电脑端显示一致
- 是否选错了电脑当前正在使用的网络地址
- 电脑端程序是否已经启动

### 多显示器下笔位置不对怎么办？

先在“连接”页重新选择目标显示器，再重新连接或重新开始使用。

### 压感手感不合适怎么办？

先试试预设：

- 太轻：尝试 **扎实**
- 太重：尝试 **轻柔**
- 想保持中性：使用 **线性**

如果仍不满意，再手动拖动曲线控制点。

### 快捷键录错了怎么办？

可以直接重新录入；如果改得太多，也可以使用“恢复默认”。
